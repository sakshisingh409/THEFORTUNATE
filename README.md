# THE FORTUNATE – NGO Management Platform

THE FORTUNATE is a full-stack web application developed to help NGOs efficiently manage donations, volunteers, campaigns, and community outreach through a modern, user-friendly interface.

The platform streamlines NGO operations by providing secure authentication, role-based access, and centralized data management using Supabase as the backend and database.

## ✨ Features

- 🔐 Secure user authentication with Supabase Auth
- 👤 Role-based access for Admin and Users
- 💰 Donation and campaign management
- 🤝 Volunteer registration and management
- 📢 Event and awareness campaign updates
- 📱 Fully responsive and modern UI
- ☁️ Real-time database integration with Supabase
- ⚡ Fast and scalable architecture

## 🛠️ Tech Stack

### Frontend
- React.js
- TypeScript
- Tailwind CSS
- Vite

### Backend
- Supabase
- PostgreSQL Database
- Supabase Authentication

## 📂 Project Structure

## 🎯 Objective

The goal of THE FORTUNATE is to simplify NGO management by providing a secure, scalable, and efficient platform that connects organizations with volunteers and donors while improving operational transparency.
